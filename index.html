Projeto Python

Projeto para disciplina de Python, utilizando o framework Django. Neste projeto foi pedido para criarmos um gerenciamento de reservas de salas empresarias para reuniões diversas, com o intuito de facilitar o trabalho da secretária. Aproveitei a interface que o django oferece e dei um encremento a mais no projeto para facilitar o entendimento do usúario final.

Projeto Java

Projeto para disciplina de Java Orientado a Objetos, utilizei o Netbeans e sua interface. No projeto foi pedido para gerenciar as salas e auditórios da faculdade do Iesp. Onde o uso da interface era obrigatória para entendimento dos alunos e aprimoramento do conhecimento. Com o projeto finalizado o usúario final iria poder adicionar, reservar, excluir qualquer sala e auditório, ver se já tinha sido reservado, entre outras funcionalidades.
